Informations de Crowdly sur GitHub


Depuis la version v.0.0.4, vous mettons en ligne chaque modifications du site Crowdly sur GitHub grâce à un contrôle de code souce (Git). Nous mettons aussi en place les "Issues", les "Pull Requests" dans notre repositories sur GitHub.


Il est nécéssaire d'avoir un compte GitHub pour suive notre repositories, le lien sera affiché en bas de ce texte. Cependant, si vous vous détectez un problème de compabilité, des fichiers tels que les images, vidéos..., des problèmes de liens, des fautes d'ortographes et etc..., vous pouvez nous les envoyer dans l'onglet "Issues" dans notre repositories Crowdly sur GitHub.
